# 内外装仕様一覧表メーカー

住宅の内外装仕様書類を「選ぶだけ・埋めるだけ → きれいに印刷／PDF」で作成できるブラウザアプリです。
新人の方でも作成できることを目標にしています。

**作成元**: アイニコグループ株式会社／楓工務店

## 作れる書類（5タブ）

| タブ | 内容 |
|---|---|
| 仕様一覧表 | 外壁・玄関ドア・屋根・設備・クロス等の仕上げ仕様 |
| 建具表 | 1階／2階の建具一覧（種類・品番・色・サイズ・把手など） |
| クロス図面 | 部屋ごとの壁・床・天井のクロス／クッションフロア指定 |
| 照明図面 | 照明器具の配置と品番（AI価格抽出機能あり） |
| 配線数量表 | 1階／2階の配線・スイッチ数量集計 |

それぞれ印刷／PDF出力に対応。お客様署名欄・承認書欄つき。

## セットアップ

### 1. 必要なもの

- Python 3.10 以上
- Anthropic API キー（OCR機能を使う場合のみ。発行は [https://console.anthropic.com/](https://console.anthropic.com/)）

### 2. 取得とインストール

```powershell
# このリポジトリを clone
cd "$env:USERPROFILE\claude_workspace"
git clone https://github.com/kawaichika-tech/naigaisou-app.git
cd naigaisou-app

# 依存ライブラリをインストール
pip install -r requirements.txt
```

### 3. APIキーの設定（OCR機能を使う場合のみ）

`.env.example` をコピーして `.env` という名前で保存し、APIキーを設定します。

```powershell
Copy-Item .env.example .env
notepad .env
```

メモ帳で `ANTHROPIC_API_KEY=ここにあなたのAPIキーを貼り付け` の右側をAPIキーに置き換えて保存。

> ⚠ `.env` は `.gitignore` で除外されています。GitHub等に絶対に上げないでください。

## 起動方法

### 方法A: Streamlit経由（推奨・黒画面なし）

```powershell
streamlit run streamlit_app.py
```

→ ブラウザが自動で開き [http://localhost:8501](http://localhost:8501) で表示されます。
OCRサーバーもバックグラウンドで自動起動するので、黒画面（コマンドプロンプト）は出ません。

### 方法B: 直接HTMLを開く（シンプル）

`仕様一覧表メーカー.html` をダブルクリックでブラウザに表示されます。
OCR機能を使う場合は別途 `OCR起動.bat` をダブルクリック（黒画面が開きます）。

## 使い方

- 上部のタブで書類を切り替え（仕様一覧表／建具表／クロス図面／照明図面／配線数量表）
- 「編集／プレビュー」で表示モード切り替え
- 「🖨 印刷/PDF保存」で表示中の書類を印刷
- 「💾 物件を保存／📂 読込」で全書類をまとめて1ファイル（JSON）で保存・読込

入力内容はブラウザに自動保存されます（取り込み画像が多いと容量警告が出る場合あり → そのときはファイル保存で対応）。
